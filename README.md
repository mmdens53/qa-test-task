# QA Test Task (Manual Testing)

Учебный QA-проект, выполненный в рамках тестового задания на позицию QA Intern / Junior QA.

## Цель проекта
Продемонстрировать подход к тест-анализу, декомпозиции требований,
формированию тестовых сценариев, оформлению баг-репортов и анализу спецификаций.

## Что представлено в репозитории

- `test_task_description` — описание каждого задания
- `checklist_cart.md` — чек-лист проверки функциональности корзины интернет-магазина  
- `test_ideas_registration.md` — идеи проверок формы регистрации  
- `bug_report_registration.md` — пример оформленного баг-репорта  
- `logic_analysis.md` — анализ логики функции и поиск потенциальных проблем  
- `api_spec_review.md` — анализ спецификации микросервиса и найденные несоответствия  
- `sql_queries.md` — SQL-запросы для работы с базой данных интернет-магазина

## Инструменты и подход
- Ручное тестирование
- Тест-анализ и декомпозиция
- Черный ящик
- Аналитическое мышление
- SQL (базовый уровень)

Проект выполнен в учебных целях.
