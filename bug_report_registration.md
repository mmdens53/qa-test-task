# Баг-репорт: кнопка «Зарегистрироваться» не выполняет действие

**Заголовок:** Кнопка «Зарегистрироваться» не выполняет регистрацию при корректно заполненной форме.

**Шаги воспроизведения:**
1. Перейти на страницу регистрации.
2. Заполнить поля формы:
   - Имя: Иван
   - E-mail: qa_tester@yandex.ru
   - Пароль: 1234Qaz4321
   - Подтверждение пароля: 1234Qaz4321
3. Нажать кнопку «Зарегистрироваться».

**Фактический результат:**  
Регистрация не выполняется, страница не изменяется, сообщения об ошибке или подтверждении отсутствуют.

**Ожидаемый результат:**  
Пользователь успешно регистрируется и:
- либо перенаправляется на страницу профиля / главную страницу авторизованного пользователя;
- либо видит сообщение об успешной регистрации с предложением авторизоваться;
- либо получает явное уведомление о необходимости подтвердить E-mail (если это предусмотрено).

**Окружение:**  
- ОС: macOS 15.6.1 (24G90)  
- Браузер: Firefox 20251009125714

**Серьезность:** Высокая (невозможна успешная регистрация при валидных данных).

**Дополнительная информация:**  
Ошибки валидации на форме отсутствуют.  
При анализе в Developer Tools требуется проверить наличие ошибок в консоли и корректность сетевых запросов (статусы/ответы).  
Возможные причины: отсутствие привязки кнопки к обработчику события или некорректная настройка серверного маршрута.
